# 内购挑选助手

一个用来处理每月内购商品清单的小工具。把 Excel 商品清单导入网页后，可以按品牌、类型、价位和预算筛选商品，加入购物车或候选清单，最后导出订购单或分享图，方便自己挑选。


## 主要功能

- 导入 `.xlsx`、`.xls`、`.csv` 商品清单
- 自动识别商品名、品牌、价格、库存、图片等字段
- 自动合并重复商品，减少重复挑选
- 按品牌、类型、价位、预算进行筛选
- 支持浅色、深色、奶油柔和三种界面风格
- 三栏宽度可以拖动调整，也可以一键恢复默认布局
- 商品可以加入购物车、候选清单和对比小桌
- 购物车按品牌分组，方便核对
- 可复制订购单、下载 CSV、导出分享图
- 支持自定义分类关键词，方便每月新清单继续复用

## 怎么使用

1. 打开网页。
2. 点击左侧「选择文件」。
3. 选择本月的商品清单 Excel 或 CSV 文件。
4. 根据需要筛选品牌、类型、价位和预算。
5. 点击商品卡片加入购物车，暂时不确定的可以先放进候选清单。
6. 选好后，在「输出清单」里复制订购单、下载 CSV，或者导出分享图。

## 数据和隐私

这个工具是纯前端网页，商品清单会在浏览器本地读取和处理。

也就是说：

- 导入的 Excel 不会自动上传到服务器
- 购物车、候选清单、预算、主题等偏好会保存在当前浏览器本地
- 换电脑或换浏览器后，本地保存的数据不会自动同步

## 常见问题

### 导入 Excel 后图片不显示

图片能否显示取决于清单里的图片链接是否允许外部访问。如果原图链接需要登录、过期了，或者禁止外部网页读取，就可能无法显示。

### 分类不准确怎么办

可以在左侧最底部的「其他操作」里展开「分类维护」，给未识别商品补充关键词。补充后的关键词会保存在当前浏览器本地，下次导入清单时继续生效。

## 适合谁用

适合每月面对一大张内购商品清单、需要控制预算、筛选品牌、整理订购单的人使用。

尤其适合：

- 自己慢慢挑
- 和同事一起核对
- 给朋友发一张确认图
- 每月复用类似格式的商品清单
